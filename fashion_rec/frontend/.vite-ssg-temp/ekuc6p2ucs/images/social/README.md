# Social Media Icons

Place the social media icon assets in this directory:

- `twitter.png` - Twitter icon
- `facebook.png` - Facebook icon
- `instagram.png` - Instagram icon
- `youtube.png` - YouTube icon
- `linkedin.png` - LinkedIn icon

## Image Requirements

- Format: PNG (transparent background) or SVG
- Size: recommended 64x64 px or larger (2x resolution)
- Style: color or monochrome icons

## Recommended Resources

Free icon sources:
- [Flaticon](https://www.flaticon.com/)
- [Icons8](https://icons8.com/)
- [Font Awesome](https://fontawesome.com/)
- [Simple Icons](https://simpleicons.org/)
